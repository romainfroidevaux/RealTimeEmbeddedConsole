enum imx27_clock_sources

EIA-FR - APF-27 Development Environment for LMI Labs.

Abstract: APF27 Clocking Module for newlib system calls

Purpose: This module provides several methods to deal with the i.MX27 imx27_clocking module Definition of the imx27_clock sources which are used by i.MX27 internal peripherals.
